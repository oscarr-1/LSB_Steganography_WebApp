Image Steganography

Hide Your Own Message

Please enter a message and an image file of any format. The message will be converted into binary and hidden within the least significant bits of the pixels. A new JPEG image holding the message will be created and available for download. If you compare the original and modified images, you will not be able to tell the difference between the two.


Reveal A Hidden Message

Please upload an image file that has a hidden message. This will extract messages hidden within images via the least signifcant bit technique.
