Audio Steganography

Hide Your Own Message

Please enter a message and an audio file in .wav format. The message will be converted into binary and hidden within the least significant bits of the audio frames. A new WAV audio file holding the message will be created and available for download. If you compare the original and modified audio, you will not be able to tell the difference between the two.


Reveal A Hidden Message

Please upload an audio file (.wav) that has a hidden message. This will extract messages hidden within WAV files via the least signifcant bit technique.
